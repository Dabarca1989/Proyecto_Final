INSTRUCCIONES PARA EJECUTAR EL PROGRAMA:

1. Clonar el proyecto
2. Instalar SQL Server Express y SSMS
3. Conectarse a .\SQLEXPRESS
4. Ejecutar el script DB_Project_2.sql
5. Verificar que exista la base SistemaReparaciones
6. Revisar que Web.config use:
   Data Source=.\SQLEXPRESS;Initial Catalog=SistemaReparaciones;Integrated Security=True
7. Ejecutar el proyecto
